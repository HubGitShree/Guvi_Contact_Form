## Overview

### The challenge



- You are required to build and submit the Guvi.in contact form using TypeScript.
- The form should contain the following fields:
  1. **Name**
  2. **Email**
  3. **Contact Number**
  4. **Subject**
  5. **Message**
  6. **Send Button**
